How To Play?

Eight Colors is a block shifting game. The Goal? Shift circular blocks to reach the target given.

Controls
1. Move the row or column by pressing W/A/S/D or Arrow Keys.
2. To select which row/column to move, you can either use Shift + W/A/S/D or Arrow Keys or click on the circular block, which will select its row + column.
Controls
To shift a circular block, simply swipe in that direction over the row or column.

Note: The game resumes right from where you left. So don't worry about closing you window. If you open the game again (in the same browser and device), it will start from the point you left.
